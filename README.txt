StudyMess — это мессенджер для обмена информацией, реализованный на языке C#. Проект состоит из двух частей: серверной (ASP.NET Core Web API) и клиентской (WPF-приложение).

Возможности:
- Регистрация и аутентификация пользователей (JWT)
- Создание и управление чатами (групповые и личные)
- Отправка текстовых и файловых сообщений
- Онлайн-статусы пользователей
- Админ-панель для управления пользователями и группами
- Хранение файлов на сервере
- Реализация SignalR для обмена сообщениями в реальном времени

Технологии:
- Backend: ASP.NET Core 9, Entity Framework Core, JWT, SignalR, SQLite
- Frontend: WPF (.NET 9)
- Прочее: DI, REST API, OpenAPI (Swagger), BCrypt.Net

Как запустить проект:
1. В репозитории проетка нажмите <>Code;
2. Выберите "Открыть в Visual Studio 2022";
3. Выполняйте дальше указания по серверу и клиенту.

Сервер:
1. Выберите проект `StudyMess_Server` как запускаемый в Visual Studio 2022;
2. Проверьте адрес в `launchSettings.json';
3. Запустите сервер;
4. Сервер будет доступен по адресу, указанному в `launchSettings.json`, например `https://localhost:7267`.(для просмотра Swagger перейдите в https://localhost:7267/swagger).

Клиент:
1. Откройте проект `StudyMess_Client` как запускаемый в Visual Studio 2022;
2. Убедитесь, что в `UrlStorage.cs` указан актуальный адрес сервера;
3. Запустите клиентское приложение;
4. Зарегистрируйтесь или войдите в систему под администратором(данные от администратора: login - admin, password - adminqpol);
5. Создайте чат или войдите в существующий;
6. Общайтесь, отправляйте сообщения или файлы.

Конфигурация
- Адрес сервера указывается в `launchSettings.json` (сервер) и в `UrlStorage.cs` (клиент).
- Обязательно использовать одинаковые значения URL в обоих местах.
